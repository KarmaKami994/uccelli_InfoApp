## ğŸ›  Getting Started

### Prerequisites

- [Flutter SDK](https://flutter.dev/docs/get-started/install) (â‰¥3.10.x)  
- Android SDK & Command-line Tools  
- A Firebase project (for push notifications)  
- (Optional) Xcode & CocoaPods for iOS support  

### Installation

1. **Clone this repo**  
   ```bash
   git clone https://github.com/KarmaKami994/uccelli_InfoApp.git
   cd uccelli_InfoApp
   ```

2. **Fetch packages**  
   ```bash
   flutter pub get
   ```

3. **Configure Firebase**  
   - Install FlutterFire CLI:  
     ```bash
     dart pub global activate flutterfire_cli
     ```
   - Run configure (select Android, iOS, Web as needed):  
     ```bash
     flutterfire configure --project your-firebase-project-id
     ```

4. **Android setup**  
   - Place your `google-services.json` in `android/app/`.  
   - Accept Android SDK licenses:  
     ```bash
     flutter doctor --android-licenses
     ```
